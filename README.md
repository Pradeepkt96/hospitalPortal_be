# hospitalPortal_be
A secure and scalable full-stack web application designed for managing healthcare interactions between patients and providers. The project integrates a React.js frontend with a Node.js (Express) backend and a SQL database (PGSQL) and for test cases Karma. Authentication and role-based access ensure data privacy and proper access control.
